I will implement a **Navbar Menu Editor** in the admin panel to allow full control over the public navigation links.

### **Plan:**

1.  **Database Migration**:
    *   Create a `navigation_items` table.
    *   Columns: `id`, `label` (e.g., "Home", "Works"), `url` (e.g., "#home", "/about"), `is_active`, `sort_order`.

2.  **Model Creation**:
    *   Create `App\Models\NavigationItem`.

3.  **Admin Panel - Navigation Management**:
    *   Create a Livewire component `App\Livewire\Admin\Navigation\Index` (and view).
    *   Features: Add, Edit, Delete, Toggle Active, Reorder.
    *   Add "Navigation" to the Admin Sidebar.

4.  **Frontend Integration**:
    *   Update `layouts.public.blade.php` to fetch `NavigationItem` records dynamically.
    *   Replace the hardcoded mobile menu links with a loop over the active navigation items.
    *   Ensure the styling remains consistent with the current full-screen overlay menu.

This gives you the flexibility to change menu labels, add new external links, or hide sections without touching the code.
